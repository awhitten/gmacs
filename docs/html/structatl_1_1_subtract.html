Detailed Description

template<class REAL_T, class LHS, class RHS>
struct atl::Subtract< REAL_T, LHS, RHS >

Expression template to handle subtraction.

\( f(x) - g(x) \)

or

\( f_{i,j}(x) - g_{i,j}(x) \)
