template<class REAL_T, class LHS, class RHS>
struct atl::Pow< REAL_T, LHS, RHS >

Expression template to handle pow for variable or container expressions.

\( f(x)^{g(x)} \)

or

\( f_{i,j}(x)^{g_{i,j}(x)} \)
