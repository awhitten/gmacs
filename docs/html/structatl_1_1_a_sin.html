Detailed Description

template<class REAL_T, class EXPR>
struct atl::ASin< REAL_T, EXPR >

Expression template to handle arcsine for variable or container expressions.

\( \arcsin f(x,y) \)

or

\( \arcsin f_{i,j}(x,y) \)
