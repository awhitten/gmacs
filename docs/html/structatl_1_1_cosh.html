Detailed Description

template<class REAL_T, class EXPR>
struct atl::Cosh< REAL_T, EXPR >

Expression template to handle hyperbolic cosine for variable or container expressions.

\( \cosh f(x,y) \)

or

\( \cosh f_{i,j}(x,y) \)
