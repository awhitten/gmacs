template<class REAL_T, class LHS, class RHS>
struct atl::Multiply< REAL_T, LHS, RHS >

Expression template to handle multiplication.

\( f(x) * g(x) \)

or

\( f_{i,j}(x) * g_{i,j}(x) \)
